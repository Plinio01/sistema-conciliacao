# 🏦 Sistema de Conciliação Bancária PLINIO ALVES BARBOSA

Sistema web para identificar diferenças entre extratos bancários e contábeis.

## Como usar

1. Faça upload do extrato bancário (CSV ou Excel)
2. Faça upload do extrato contábil (CSV ou Excel)
3. Clique em "Processar Conciliação"
4. Baixe o relatório completo

## Formato dos arquivos

Os arquivos devem conter as colunas:
- `data` (formato: AAAA-MM-DD ou DD/MM/AAAA)
- `valor` (número decimal)
- `descricao` (texto)
